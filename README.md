# portal-main-notes

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库旨在以静态 HTML 页面的形式，整理、存档并公开发布一系列独立的笔记、工具或信息页面。所有页面均为独立文件，不依赖特定后端服务，可直接在浏览器中打开或通过静态托管服务访问。

## 目录说明

```
portal-main-notes/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各个独立 HTML 页面的目录
│   ├── example.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/`：核心目录，每个 HTML 文件代表一个独立的归档页面。
- `assets/`：存放多个页面可能共用的样式、脚本或图片资源。

## 页面归档说明

- 每个 HTML 页面应尽量自包含，资源引用建议使用相对路径或指向 `assets/` 目录。
- 页面内容可以是笔记、小工具、信息汇总等，不限制具体主题。
- 新增页面时，建议在 `pages/` 下创建新文件，并在 `index.html`（若存在）中更新导航链接。

## 维护说明

- 欢迎通过 Pull Request 提交新的归档页面或改进现有页面。
- 提交前请确保页面无外部依赖（或已明确说明），且内容不违反平台使用条款。
- 本仓库不承诺长期维护，但会尽力保持页面可访问。
